# Portfolio [🔗](https://yoginiaishwaryaa.github.io/portfolio-pts/)

A modern, responsive portfolio website built with React, TypeScript, and Vite.

## Tech Stack

- **Framework**: React 18
- **Language**: TypeScript
- **Build Tool**: Vite
- **Styling**: CSS

## Features

- Interactive UI with custom cursor
- Animated mascot character
- Navigation menu
- Hero section
- Projects showcase
- Publications section
- Skills display
- Contact information
